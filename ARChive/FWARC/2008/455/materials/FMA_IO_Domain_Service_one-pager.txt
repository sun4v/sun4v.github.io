2. Project Summary
    2.1. Project Description:

	This project defines a Fault Management Architecture (FMA) domain
	service required in a logical domain environment where ownership of one
	or more PCI root complexes is removed from the control domain and
	assigned to so-called root domains. This project extends the set of FMA
	Domain Services originally specified in FWARC/2006/141.
